# Online Shopping System (Advanced Java)

## Description
Online shopping web application developed using
Servlets, JSP, and JDBC.

## Technologies Used
- Java
- Servlets
- JSP
- JDBC
- Oracle Database
- Apache Tomcat

## Features
- Admin login
- Customer registration & login
- Product add, update, delete
- View and buy products

## Deployment
Packaged as WAR and deployed on Apache Tomcat.

## How to Run
1. Import the project into Eclipse as a Dynamic Web Project
2. Add Oracle JDBC driver (ojdbc11.jar) to WEB-INF/lib
3. Configure database details in DBConnect class
4. Deploy the project as a WAR on Apache Tomcat
5. Access the application using browser
